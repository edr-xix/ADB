# ADB & Fastboot Manager

A cross-platform graphical utility for managing Android devices via ADB and Fastboot. Built with a focus on accessibility, this tool provides a visual interface for device filesystems alongside an integrated, screen-reader-friendly command terminal.

## Core Features

* **Integrated Terminal:** Execute raw ADB and Fastboot commands directly within the application.
* **Accessible Output:** Native text-to-speech integration that automatically dictates terminal output, engineered specifically for accessibility.
* **Visual File Manager:** A full graphical interface to navigate the Android filesystem. Push and pull files to and from the host machine, create or delete directories, and move files internally across the device.
* **Targeted Device Selection:** Built-in device polling allows you to actively select the target device in multi-device environments, ensuring commands and file transfers execute on the correct hardware.
